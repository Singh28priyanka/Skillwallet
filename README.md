
# ShopEZ – E-Commerce Platform

ShopEZ is a full-stack e-commerce application where users can browse products,
add items to cart, and place orders securely.

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express.js

Database
- MongoDB

## Features

✔ User Authentication  
✔ Product Listing  
✔ Shopping Cart  
✔ Order Management  
✔ Admin Dashboard

## Project Architecture
MVC Architecture
